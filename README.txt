1. Compilação
-------------

	Para compilar o EP utilize o Makefile:
	
		$> make

	Também é possível compilar o programa utilizando diratamente o gcc: 	

		$> gcc ep1.c -o ep1 -Wall -ansi -pedantic -lpthread


2. Utilização
-------------

	Para utilizar basta informar o arquivo de entrada:

		$> ./ep1 entrada1.txt
